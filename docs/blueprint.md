# **App Name**: EstateFlow CRM

## Core Features:

- Add Contact Form: Form to input contact details: Name, Email, Phone, Budget (dropdown), and Status (dropdown).
- Automatic Serial Numbering: Automatically assign a unique serial number (N1, N2, N3, etc.) to each contact upon submission.
- Contact List View: Display contacts in a sortable table with columns for Serial Number, Name, Phone, Status, and Budget.
- Sortable Contact List: Sorting functionality based on Budget, Status, or Name columns.
- Quick Contact Buttons: Generate WhatsApp link (using the `Hey` message).  Button opens default mail client, pre-filled with the contact's email address.
- Edit Contact Module: Module to edit existing contact information, pre-filling existing values in a web form.
- Lead Scoring Assistant: AI Tool to identify high-potential leads (Hot status) from interaction data. It flags communications of interest and generates recommended questions for sales.

## Style Guidelines:

- Primary color: A professional blue (#2E9AFE) for trust and reliability.
- Background color: Light blue (#EBF5FF), providing a clean and calming backdrop.
- Accent color: A vivid orange (#FF851B) to highlight key actions, standing out against the cool blues to draw attention to CTAs.
- Body and headline font: 'Inter', a sans-serif font offering a clean, objective look.
- Use clear, minimalist icons for navigation and actions (e.g., edit, delete, send message).
- Clean, intuitive layout with clear sections for adding, viewing, and editing contacts.
- Subtle transitions for actions like opening contact details or saving changes.
